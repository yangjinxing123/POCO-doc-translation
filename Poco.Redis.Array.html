Poco::Redis

class Array

Library: Redis
Package: Redis
Header: Poco/Redis/Array.h

Description

Represents a Redis Array. An Array can contain Integers, Strings, Bulk Strings, Errors and other Arrays. It can also contain a Null value.

Known Derived Classes: Command

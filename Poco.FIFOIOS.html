class FIFOIOS

Library: Foundation
Package: Streams
Header: Poco/FIFOBufferStream.h

Description

The base class for FIFOBufferInputStream and FIFOBufferStream.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.

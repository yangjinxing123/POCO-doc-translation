createRedisType

static RedisType::Ptr createRedisType(
    char marker
);

Create a Redis type based on the marker:

  • '+': a simple string (std::string)
  • '-': an error (Error)
  • '$': a bulk string (BulkString)
  • '*': an array (Array)
  • ':': a signed 64 bit integer (Int64)

Poco::Net

class MailRecipient

Library: Net
Package: Mail
Header: Poco/Net/MailRecipient.h

Description

The recipient of an e-mail message.

A recipient has a type (primary recipient, carbon-copy recipient, blind-carbon-copy recipient), an e-mail address and an optional real name.

Poco::Net

class NameValueCollection

Library: Net
Package: Messages
Header: Poco/Net/NameValueCollection.h

Description

A collection of name-value pairs that are used in various internet protocols like HTTP and SMTP.

The name is case-insensitive.

There can be more than one name-value pair with the same name.

Inheritance

Known Derived Classes: HTTPMessage, HTTPServerResponse, HTTPServerResponseImpl, MailMessage, MessageHeader, HTTPAuthenticationParams, HTTPRequest, HTTPServerRequestImpl, HTMLForm, HTTPResponse, HTTPServerRequest
